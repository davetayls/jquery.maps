jQuery.maps: inserting Google Maps from markup

jQuery.maps is an accessible way to implement Google maps using markup. You don't even have to load Google Maps api.

  • without pins
  • basic pins
  • custom pins

Location Name: 13.251, 123.684. Zoom: 12 

<div id="map">
    <h2 class="maps-title">Map Details: <a href="#map-skip">skip</a></h2>
    <div class="geo">Location Name: 
        <span class="latitude">13.251</span>, 
        <span class="longitude">123.684</span>.
        Zoom: <span class="zoom">12</span>
    </div>
    <div class="maps-container">
        <div class="maps-container-loading"></div>
    </div>
    <div id="map-skip"></div>
</div>
